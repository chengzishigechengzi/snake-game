# 🚀 阿里云轻量服务器部署指南 (深圳节点)

恭喜你选择了国内节点！这将带来最极致的游戏流畅度。
请按照以下步骤操作，全程只需要复制粘贴命令即可。

## 第一步：连接服务器
1. 登录阿里云控制台，进入【轻量应用服务器】。
2. 找到你的深圳服务器卡片，点击进入详情。
3. 点击右上角的 **“远程连接”** (Workbench) 按钮。
4. 会弹出一个黑色的命令行窗口，看到 `#` 光标闪烁就说明连接成功了。

---

## 第二步：一键部署代码 (在黑色窗口中依次执行)

请**一行一行**复制下面的命令，粘贴到黑色窗口中并回车：

### 1. 准备环境并下载代码
```bash
# 进入用户主目录 (确保有权限)
cd ~

# 安装 git (如果已安装会提示 Nothing to do)
sudo yum install -y git

# 拉取你的游戏代码
git clone https://github.com/chengzishigechengzi/snake-game.git

# 进入游戏文件夹
cd snake-game
``````

### 2. 安装依赖并启动
```bash
# 安装项目依赖 (可能需要几十秒)
npm install

# 安装后台运行工具 PM2
npm install -g pm2

# 启动游戏！
pm2 start server.js --name "snake-game"
```

> **成功标志**：如果你看到一个绿色的表格，状态显示 `online`，说明游戏已经成功启动了！

---

## 第三步：开放端口 (最重要的一步！)
阿里云默认是锁住端口的，必须手动打开才能让别人访问。

1. 回到阿里云网页控制台。
2. 点击左侧菜单栏的 **“防火墙”** (Firewall)。
3. 点击右上角的 **“添加规则”**。
4. 填写如下信息：
   - **应用类型**：自定义
   - **协议**：TCP
   - **端口范围**：`8080`  <-- 必须填这个！
   - **限制IP**：0.0.0.0/0 (保持默认)
5. 点击“确定”。

---

## 第四步：开始游戏！
现在，你的游戏已经上线了！

**游戏链接**：
`http://你的公网IP:8080`

> **在哪里看公网 IP？**
> 在阿里云控制台的服务器概览页，有一行写着 **“公网 IP”** (Public IP)，通常是一串数字，比如 `47.106.xx.xx`。

**怎么发给朋友？**
直接把上面的链接（如 `http://47.106.12.34:8080`）复制发到微信群，大家点开就能直接玩，**不需要下载、不需要备案，完全流畅！**
